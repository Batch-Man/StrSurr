
# StrSurr - by Kvc
## Description
This program can Extract the string surrounded by given characters.

Author: Kvc

## Usage
Call StrSurr [Surrounding Chars] [String]

Example: 
`StrSurr "[]" "Hello [Kvc], how are you?"`
